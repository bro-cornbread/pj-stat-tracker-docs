The Problem

Pearl Jam has played 1,100+ shows across 35 years. Fans have no good way to track their journey.

The Collector

20+ shows. Wants to know exactly which songs they've heard, which albums they've "completed" live, and how their experience stacks up.

The Casual Fan

Goes to a show every few years. Wants to look up setlists, browse stats, and mark shows they attended.

The Stat Nerd

Loves data. Wants charts, rankings, rarity scores, trend lines, and deep breakdowns of 35 years of live history.

The First-Timer

Just went to their first show. Wants to understand what they experienced — was that setlist special? Were any songs rare?

Song Rarity System

Every song has a rarity score (0.0 to 1.0) based on how frequently it's played. Songs are bucketed into five tiers.

Ultra-Rare Played at fewer than 1 in 200 shows "Let Me Sleep"
Rare Played at 2-5% of shows "Dirty Frank"
Uncommon Played at ~10% of shows "Lukin"
Common Played at ~40% of shows "Do the Evolution"
Staple Played at most shows "Alive", "Black"

Screens & Features

14 screens across 5 tabs, plus detail views, search, and settings.

Stats Dashboard (Home)

  • Toggle between Band Stats and My Stats
  • Customizable widget grid with drag-and-drop
  • 10 band widgets (charts, ranked lists, progress bars)
  • 9 personal widgets including AI Insights carousel
  • Edit mode to add, remove, and reorder widgets

Shows Browser

  • Browse all 1,125 concerts
  • Horizontal year filter chips
  • "Attended" toggle filter (signed-in users)
  • Show cards with date, venue, location, song count, poster
  • Pull-to-refresh

Songs Explorer

  • Search bar with live filtering
  • Sort: A-Z, Most Played, Rarest
  • Rarity tier filter chips
  • Song rows with play count, rarity badge, cover badge

Venue Map

  • Interactive world map with all venues
  • Pin colors by show density
  • Tap pin for venue callout with show count
  • Dark-themed map matching app aesthetic

Profile

  • Avatar and user identity
  • Global stats grid (shows, songs, venues, countries)
  • Trophy progress bar with recent unlocks
  • Clickable rarity breakdown rows
  • Rarest songs seen and recent shows

Show Detail

  • Poster image, date, venue, location
  • "I Was There" attendance toggle
  • 1-5 star rating
  • Full setlist grouped by set with debut and rarity badges
  • Tappable songs navigate to song detail

Song Detail

  • Song name, rarity badge, album
  • Stats: total plays, first/last played
  • Collapsible lyrics section
  • Recent performance history with venue links

Album Detail

  • Cover art, release year, track count
  • Full track list with rarity badges and play counts
  • Streaming links (Spotify, Apple Music, etc.)
  • Production credits (collapsible)

Poster Gallery (planned)

  • 3-column grid of 1,000+ show posters
  • Year filter chips
  • Date overlay on each poster
  • Tap to navigate to show detail

Trophy Screen

  • Grid of all 39 trophies by tier
  • Unlocked: full color. Locked: dimmed with lock icon
  • Tap for detail modal with description and unlock date

Search

  • Unified search across songs, shows, venues
  • Results grouped by type with count badges
  • Tap any result to navigate to detail

Auth & Settings

  • Email/password login and registration
  • "Continue without account" for view-only mode
  • Password reset via email
  • Sign out with confirmation dialog
  • Persistent sessions across app restarts

User Stories

36 stories across 4 categories defining what the app must do.

Browse & Discover

S-01As a fan, I want to browse all Pearl Jam concerts so I can explore the band's touring history.
S-02As a fan, I want to filter shows by year so I can focus on a specific era.
S-03As a fan, I want to search for a song, show, or venue so I can find what I'm looking for quickly.
S-04As a fan, I want to see a song's rarity score so I know how special it was to hear it live.
S-05As a fan, I want to view a show's full setlist so I can relive (or preview) the concert.
S-06As a fan, I want to see which songs are played most often so I know what to expect at my next show.
S-07As a fan, I want to see the band's busiest venues so I can plan where to catch a show.
S-08As a fan, I want to see shows on a map so I can visualize the band's global reach.
S-09As a fan, I want to see a song's performance history so I know when and where it was last played.
S-10As a fan, I want to browse show posters in a visual gallery so I can appreciate the artwork.
S-11As a fan, I want to see which songs have the longest gap since being played so I can identify potential surprises.
S-12As a fan, I want to read a song's lyrics while viewing its stats.
S-13As a fan, I want to see album details with streaming links so I can listen to songs I discovered through the app.

Personal Tracking

S-14As a fan, I want to mark shows I attended so I can build my concert history.
S-15As a fan, I want to rate shows I attended so I can remember which were my favorites.
S-16As a fan, I want to see how many unique songs I've heard live so I can track my collection progress.
S-17As a fan, I want to see which album songs I've heard live so I can track album completion.
S-18As a fan, I want to see my rarest catches so I can appreciate the special moments.
S-19As a fan, I want to see a timeline of my attended shows so I can visualize my journey.
S-20As a fan, I want to see which songs I've heard the most so I know my personal staples.
S-21As a fan, I want to see how my show count compares to the total so I can set attendance goals.
S-22As a fan, I want to see how many cities, states, and countries I've seen shows in.
S-23As a fan, I want AI-generated insights about my concert patterns so I can discover things I didn't know about my own history.

Achievements

S-24As a fan, I want to earn trophies for attendance milestones so I feel rewarded for my dedication.
S-25As a fan, I want to see all available trophies so I know what I'm working toward.
S-26As a fan, I want trophies organized by difficulty tier so I can appreciate the progression.
S-27As a fan, I want to see my trophy progress on my profile so others (and I) can see my achievements at a glance.
S-28As a fan, I want special trophies tied to iconic venues (MSG, Wrigley) so attending those shows feels extra meaningful.
S-29As a fan, I want a trophy for hearing the original Ten album live so I have a tangible goal to chase.

Dashboard & Customization

S-30As a fan, I want a customizable stats dashboard so I see the data I care about most.
S-31As a fan, I want to reorder and add/remove widgets so the dashboard feels personal.
S-32As a fan, I want separate band stats and personal stats views so I can switch between exploring the catalog and reviewing my own history.

Account & Auth

S-33As a visitor, I want to browse all band statistics without creating an account so I can evaluate the app before committing.
S-34As a user, I want to reset my password via email so I can recover my account.
S-35As a user, I want my session to persist across app restarts so I don't have to log in every time.
S-36As a user, I want to sign out with a confirmation prompt so I don't accidentally lose my session.

Future Considerations

Not in scope for MVP, but on the roadmap.

  • Social features — Share setlists, compare stats with friends, leaderboards
  • Push notifications — Alerts when a rare song is played at tonight's show
  • Setlist predictions — ML model predicting tonight's setlist based on historical patterns
  • NFT badges — Blockchain-backed trophies for verifiable concert achievements
  • Concert photos — User-uploaded photos tied to attended shows
  • Apple Watch / Wear OS — Quick-glance widgets for setlist tracking during shows
  • Home screen widget — iOS/Android widget showing next show or recent stats
